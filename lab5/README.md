
# ЛР5 — Классификация анемии (scikit-learn)

**Файл:** `ЛР5_Кольцов.ipynb`

## Цель работы
Построить и сравнить модели классификации для предсказания целевого признака **`Анемия`** по показателям крови.

## Данные
Датасет `data.xlsx` (в ноутбуке): показатели ОАК + биохимия, включая целевой столбец **`Анемия`**.

Важно: в работе отдельно рассматривается сценарий **без `Ферритин`** (чтобы избежать утечки признаков и получить более реалистичную оценку качества).

## Что сделано в ноутбуке
- Предобработка данных:
  - анализ пропусков, типов, удаление служебных столбцов
  - создание `Возраст_годы` из `Возраст_дни`
  - масштабирование (`StandardScaler`)
- Визуализация:
  - диаграммы, confusion matrix
  - ROC-кривые и AUC
- Модели:
  - Logistic Regression
  - KNN
  - Decision Tree
  - Random Forest
- Оценка:
  - Accuracy, Precision, Recall, F1
  - Balanced Accuracy
  - Confusion Matrix
  - ROC AUC
- Разделение на группы:
  - парни/девушки × до 18 / старше 18
- Доп. задание:
  - регрессия `RandomForestRegressor` для предсказания **`Ферритин`**
  - MAE и R², важности признаков
