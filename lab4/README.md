## Что сделано в ноутбуке
### Часть 1 — Кластеризация на синтетических данных
- Генерация (или создание) выборок и фиксирование seed
- Масштабирование признаков (`StandardScaler`)
- Кластеризация с помощью:
  - `KMeans`
  - `AgglomerativeClustering`
  - `DBSCAN`
- Оценка качества кластеризации:
  - `silhouette_score` (коэффициент силуэта)
- Подбор числа кластеров:
  - метод локтя (используется `KElbowVisualizer`)

### Часть 2 — Кластеризация на реальном датасете (ecommerce_fraud.csv)
- Загрузка и обзор датасета
- Предобработка:
  - преобразования бинарных признаков (например Yes/No → 0/1)
  - работа с числовыми признаками (отбор numeric, исключение/учёт `fraud` как метки)
- Кластеризация и сравнение подходов:
  - разные методы масштабирования (например MinMax и др.)
  - подбор параметров (например `eps` для DBSCAN)
- Дополнительная оценка (где применимо):
  - метрики сравнения кластеров с известной меткой (`Adjusted Rand Index`, `V-measure`) — если в работе используется `fraud` как “истина”
